[Python]文件儲存unicode碼的問題

發表於 2014-11-19 | 分類於 Python

由於用網頁爬蟲抓取的資料存入json
文件都會顯示"{artist": "\u3058\u3093"}
但程式抓取都不會有亂碼的問題

但今天我Google大神一個整天
終於找到方法

儲存文件時請記得調成utf-8

內文的\n,\r,\t…等等都會不見
最終以str.replace('\n','\\n')來解決

repr包字串會跑出\x5e東西
