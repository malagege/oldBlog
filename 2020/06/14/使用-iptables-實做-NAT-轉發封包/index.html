預設閘道通常都是設定中華電信小烏龜
一般可能不會去了解原因

之前幾篇有講到 CIDR

只要超過遮罩時候，就會轉發到 預設閘道

Windows 10 和 Linux 如何轉發 IP

每一台電腦都可以擔任轉發動作，但一開始我在預設閘道怎麼沒有效果
其實都需要設定

Win 10

2、windows xp、Vista、windows7：

打开注册表：regedit.exe

打到：HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters

增加：
Value Name: IpEnableRouterData Type: REG_DWORD

Value: 1

重启机器。

linux

一般在設定 iptables forward
但前置作業記得要先做這個動作

查詢IP forward功能是否開啟
cat /proc/sys/net/ipv4/ip_forward
0代表沒有啟動
1代表已啟動

暫時啟動IP forward功能
echo 1 > /proc/sys/net/ipv4/ip_forward
重新開機就會失效

永久開啟IP forward功能
vi /etc/sysctl.conf
加上這句
net.ipv4.ip_forward = 1
立刻讓設定生效
sysctl -p
