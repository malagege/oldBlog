linux bash / shell 有關 * 星號的事情

發表於 2020-01-21 更新於 2021-06-27 分類於 Linux Disqus：

以前 ls 萬用字元真的很方便

這樣可以列出很多 *.jpg 圖片
但是有些細節沒有釐清
偶爾會遇到不能用的狀況下
今天終於徹底了解

釐清 *

就是現在目錄抓到相關檔案名稱
這也是為什麼? 原理就這麼簡單
但我這邊沒講更細
下面 shopt -s nullglob 會提到

shell array 應用

這樣看似可以用
但沒有 jpg ，就會產生 *.jpg
這邊要小心
這邊也是有辦法解決

參考:sh - Check if a file exists with wildcard in shell script - Stack Overflow

shopt -s nullglob
