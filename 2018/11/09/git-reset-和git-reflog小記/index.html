git reset 和git reflog小記

發表於 2018-11-09 更新於 2021-06-03 分類於 Git Disqus：

最近在改程式，所以我重新上一個commitgit commit --amend
但做完發現…，我忘記做git add xxx了
當下Ctrl+C都不能中斷
只能QQ:q!or:qa!

這種事當然要找一下解法

git reflog

git reflog指令可以看最近刪除commit，之後會被自動刪除
可依照commit說明去找上次的內容

這時候想看reflog內容要怎麼看??
這時候要用之前學得融會貫通XD
git checkout commit_id
這樣就能切換過去
再git diff也能看到東西

git reset

git reset --hard commit_id

這樣就檢單還原

git reset –soft
git reset –mixed
不過這指令記不時起來
