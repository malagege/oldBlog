define() vs const 常數定義
最主要的原因是底層的不同
define() 是 run-time constants 而 const 是 compile-time constants.
define() 是 global scope 或者在某個 namespace，所以不能用來定義 class-scope 的常數。
define()可以用在 if() 但 const不能用在 if()
