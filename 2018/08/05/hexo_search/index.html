hexo裝上搜尋功能(local search)

發表於 2018-08-05 更新於 2021-06-27 分類於 hexo Disqus：

最近文章越來越多了
應該是時候加入收尋功能
有時找新文章真的太麻煩了
雖然很簡單還是筆記一下

其實很簡單
簡單紀錄一下

  1. 在hexo站點目錄下執行npm install hexo-generator-searchdb --save
  2. Next主題config.yml設置 基本上這樣就可以了
