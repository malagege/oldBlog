Docker volumes 路徑

docker volumes 存在位置在/var/lib/docker/volumes需要有root權限才能進去

docker rm 並不會刪除volumes檔
想一次刪掉container跟volume嗎?
只要下這個指令docker rm -v container_id
就能玩成目地了
