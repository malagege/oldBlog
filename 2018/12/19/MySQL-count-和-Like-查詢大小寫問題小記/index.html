另外MySQL like查詢預設不分大小寫
其他DB我記得預設會分大小寫
不常寫應該很容易忘記

DB LIKE到底有沒有分大小寫呢?

MySQL , SQL Server沒有分
Oracle,DB2有分

要讓mysql查詢區分大小寫，可以：
select * from table_name where binary a like ‘a%’
