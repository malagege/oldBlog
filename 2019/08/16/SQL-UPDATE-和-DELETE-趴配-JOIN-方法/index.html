SQL UPDATE 和 DELETE 趴配 JOIN 方法

發表於 2019-08-16 更新於 2021-06-27 分類於 MySQL Disqus：

最近看到 UPDATE 和 DELETE
可以搭配用 JOIN 方式
這種常常查詢資料都要搭配程式 for 迴圈實作
我覺得非常實用

我網路隨便找一個範例
SQL Fiddle

UPDATE … INNER JOIN

透過 JOIN 方式可以同時改兩張表的資料
但還有更神奇的妙用
就是把b表換成子查詢
可以用 JOIN 方式改變a表的內容
省了寫程式 code
超級實用

DELETE … INNER JOIN

DELETE 也能搭配 JOIN
不過有點跟原本 DELETE 不太一樣

如下

通常穩們用的 DELETE 都是改from表
但 JOIN 不是這樣，是針對 DELETE 後面 table 命名 a,b 做刪除
當然也可以不用刪除兩張表
DELETE a FROM ...
或
DELETE b FROM ...
都可以指定刪除

此招也很實用
