DataLoader

目前查到資料可以用 DataLoader 方式解決問題

Dataloader 主要有兩項功能： Batching (批次) & Caching (快取) 。
Batching 在於他能夠將想要進 db 搜尋的 id 都搜集起來，等時間到了一次進 db 搜尋，解決 N + 1 問題。
Caching 在於它在紀錄 id 時會做 memoization ，所以未來若是有重複的 id 進來就會被剔除，保證資料索取數越少越好。
