docker 修改經驗小記

  1. RUN 這個指令 假如是跟上面有關的，記得要用以下方法
  1. 安裝指令，記得要用apt-get update && apt-get install -y --no-install-recommends \

  1. 要合併 images ，建議找源頭一樣 image，這樣編譯 Dockerfile 比較無痛

  2. Slim,Stretch,Alpine 那些到底是什麼?
