產生方法

  1. 在 Sublime 使用 regex ((.*\n){1,3})
  1. 選擇 Find all 按鈕
  1. 按 「Shift + 方向鍵左(←)」 後，再按「Shift + 方向鍵左(↑)」

  1. Ctrl + J ，按完就完成兩段句子，最後一行可刪掉

  2. 複製兩段 LIKE SQL 語句，中間替代參數也要先設定

  1. 全部框選:id: Ctrl + D，把1,2行複製起來，貼上即可完成
  1. 把最後,移除

就可以快樂貼到編輯器查詢。
